### 📜 What is Pond Mirror?

This is the serene public reflection of **TobyWorld** — a peaceful glassmorphic space built for the ToadGang.

It consists of two main pages:
- **index.html** — The main Pond Mirror: a reflective landing page with elegant tag-style navigation and the mantra *"Patience rewarded"*.
- **scrolls.html** — The living archive containing **50 eternal whispers** (lore cards) with live search, tag filters, smooth modals, and calm interactions.

Built with pure HTML, CSS & JavaScript. Fully responsive, with dark/light mode toggle and soft glassmorphic aesthetics.
